FOTOS E IMAGENS

Nessa parte do curso foi ensinado como se colocar uma imagem em nosso site. Para simplificar esse processo, irei resumir esse capitulo em alguns passos.

1° PASSO: Procurar a imagem correta

A principio, nos precisamos antes de tudo ter alguma imagem, para isso nos precisamos de imagens que tenham seus direitos autorais liberados para ser usadas com cunho comercial. O Google é uma exelente alternativa, já que o mesmo nos fornece filtros que podemos colocar em nossas pesquisas para obter essas fotos

2° PASSO: Baixar o programa GIMP

Agora que já temos a foto, nos precisamos fazer umas edições basicas nela. Não vai ser preciso mexer em nada relacionado a seu design, ou algo do tipo. Essa edição na verdade vai ser para deixar a nossa imagem em dimensões ideais para o nosso site, contribuindo não só para uma vizualização mais limpa, como tambem para reduzir o "peso" da nossa imagem. tornando assim o nosso site mais leve

3° PASSO: Colocar a foto no site

Agora vem a parte mais importante. Após fazer todas as alterações em nossa foto, vamos colocar ela em nosso site. Primeiro, pra facilitar a nossa vida, precisamos baixar essa foto na mesma pasta que está o programa que vai recebe-la. Feito isso, é so abrir o programa e dar o comando "img", apos isso vamos utilizar as teclas Ctrl + Espaço, e então o arquivo da nossa foto estara la e é so seleciona-la. Para dar um exemplo, irei adicionar duas imagens em nosso site

TAGS E FORMATAÇÕES

NEGRITO

Um texto em negrito com a tag "B", é diferente de um texto em negrito com a tag "STRONG"
Ex: Esse texto está em negrito com a tag "strong"
Esse texto está em negrito com a tag "B"

Visivelmente não tem diferença nenhuma em sua forma. Porem, para alguns sites, a forma mais atualizada (strong) é mais aceita

ITALICO

Um texto em italico com a tag "i", é diferente de um texto em italico com a tag "em"
Ex: Esse texto está em italico com a tag "I"
Esse texto está em italico com a tag "EM"

Visivelmente não tem diferença nenhuma em sua forma. Porem, para alguns sites, a forma mais atualizada (em) é mais aceita

CITAÇÃO

Como ja foi aprendido durante o curso, o HTML5 preza pelo significado das coisas, e quando utilizamos a tag "q", estamos informando para o sistema que o que está dentro da tag é a citação de alguem. Como ex:
Segundo o filosofo Aristóteles: O prazer no trabalho aperfeiçoa a obra.

TAG <blockquote>

Não é muito diferente da tag "q", porem essa tag é mais completa e é muito utilizada para citações de documentos, obras, livros, passagens de site, entre outros. Pois nessa tag, você tem a opção de informar o link de origem para o sistema. Esteticamente não muda nada em seu site, mas o sistema entende que aquilo é uma citação e de onde ela vem. Basta usar a tag "blockquote" + o complemento "cite" Como ex:
De acordo com o site, guiadacarreira:

Existe o mito de que a produtividade só é importante para quem quer trabalhar muito. Mas isso é um engano.

A tag vai ficar da seguinte forma:
<blockquote cite=" link "> citação </blockquote>

TAG <abbr>

Para finalizar.Essa tag serve para definir uma abreviação. Ex:
A Dra. vai atender o paciente

Se passarmos o cursor do mouse em cima da sigla "Dra." vai aparecer o nome sem abreviação

LISTAS OL E UL

LISTAS ORDENADAS

Lista ordenada serve literalmente para fazer uma lista em ordem crescente, seja ela alfabetica, numerica, ou em numeros romanos. Para usa-la vamos usar duas tags. A primeira é "<ol>", essa tag tem a função de avisar ao sistema que estamos fazendo uma lista. Dentro da tag "<ol>" vamos utilizar a tag <li> dentro de cada item da nossa lista. Ex:

  1. Arrumar a casa
  2. tomar banho
  3. ir pra faculdade

Dentro da tag "ol" podemos colocar o parametro type para alterarmos o seu estilo de ordem, fica mais ou menos assim o comando: <ol type="">. Na lista abaixo vou colocar tudo em ordem alfabetica utilizando o parametro "type"

  1. Arrumar a cama
  2. Tomar banho
  3. Ir para faculdade

Além dessas funções. caso você não queira que sua lista comece pelo numero 1, podemos fazer essa alteração atravez do parametro start dentro tbm da tag "ol", ficando dessa forma:<ol start="5">. Na lista abaixo, ela vai começar pelo número 5. Ex:

  1. Arrumar a casa
  2. Tomar banho
  3. Ir para faculdade

LISTAS NÃO ORDENADAS

Praticamente igual as listas ordenadas, o estilo de lista "não ordenada" tem a mesma função da lista ordenada, sua unica diferença é que ela não tem ordem, não é necessario uma ordem de sequencia. Para reproduzi-la utilizamos praticamente os mesmos comandos da lista ordenada. Porem, no lugar de "ol", vamos usar a tag "ul". Acompanhe no exemplo abaixo:

Lista de compras
  • Pão
  • Ovo
  • Queijo
  • Frutas

LINKS INTERNOS

Para adicionar um link interno é muito facil. Primeiro precisamos ter em mente que para isso acontecer, todas as paginas/links do nosso site precisam estar em uma unica pasta. Para dar o comando, nos vamos utilizar a mesma tag para links externos: <a>. Após isso, nos damos o comando "ctrl + espaço" no local onde seria adicionado o nosso link e selecionamos a pagina que desejamos que seja aberta. Para dar um exemplo pratico, clique em mim para acessar a pagina onde ensino a colocar links externos

PARAMETROS

Assim como no Links externos, nos podemos utilizar alguns parametros de valor mais semantico para guiar o Google e ajuda-lo a entender o caminho das paginas. para isso podemos utilizar dois parametros:

REL NEXT
Usamos essa tag para informar que o link que foi utilizado por ela, é a "proxima pagina" que queremos que o usuario acesse. Lembrando que esse parametro é de valor semantico, esteticamente não muda nada em seu site
REL PREV
Ao contrario de "next", o parametro "prev" faz referencia a "pagina anterior". Seria meio que um aviso ao Google que o link que foi utilizado com esse parametro representaria a pagina que teria que voltar apos o acesso

LINKS PARA DOWNLOADS

Para criar um link para download utilizamos a mesma tag de sempre: <a>. Uma observação importante a ser feita é que o arquivo que você quer que seja baixado tem que está tambem na mesma pasta do seu site. Assim como nos links internos e externos, os links para download tem parametros que facilitam no processo, como exemplo o parametro (download="nome do arquivo que vai ser baixado") pra acompanhar a linha de raciocinio, clique em mim para baixar um PDF

OUTRAS FORMATAÇÕES

TEXTO EM FORMATO DE CODIGOS

De forma direta, essa tag nada mais faz do que deixar o espaçamento entre as letras de forma convergente. É muito utilizados pra expressar codigos. Como ex:
var a = 2
var b = 5
var x = b*a
var x = 10

TAG <PRE>

Ultilizado para replicar exatamente o que foi colocado dentro dele, contando com espaçamento, qubras de linha (enter), é muito utilizado dentro da tag "code", para expressar formulas sem ter que ficar utilizando quebras de linha. Como ex: 

        <code>
            <pre>        
        5 + 3 * (4-2) = x 
        5 + 3 * 2 = x 
        5 + 9 = x 
        x = 11
            </pre>
        </code>

CITAÇÃO

Como ja foi aprendido durante o curso, o HTML5 preza pelo significado das coisas, e quando utilizamos a tag "q", estamos informando para o sistema que o que está dentro da tag é a citação de alguem. Como ex:
Segundo o filosofo Aristóteles: O prazer no trabalho aperfeiçoa a obra.

TAG <blockquote>

Não é muito diferente da tag "q", porem essa tag é mais completa e é muito utilizada para citações de documentos, obras, livros, passagens de site, entre outros. Pois nessa tag, você tem a opção de informar o link de origem para o sistema. Esteticamente não muda nada em seu site, mas o sistema entende que aquilo é uma citação e de onde ela vem. Basta usar a tag "blockquote" + o complemento "cite" Como ex:
De acordo com o site, guiadacarreira:

Existe o mito de que a produtividade só é importante para quem quer trabalhar muito. Mas isso é um engano.

A tag vai ficar da seguinte forma:
<blockquote cite=" link "> citação </blockquote>

TAG <abbr>

Para finalizar.Essa tag serve para definir uma abreviação. Ex:
A Dra. vai atender o paciente

Se passarmos o cursor do mouse em cima da sigla "Dra." vai aparecer o nome sem abreviação

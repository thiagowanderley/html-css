LINKS INTERNOS

Para adicionar um link interno é muito facil. Primeiro precisamos ter em mente que para isso acontecer, todas as paginas/links do nosso site precisam estar em uma unica pasta. Para dar o comando, nos vamos utilizar a mesma tag para links externos: <a>. Após isso, nos damos o comando "ctrl + espaço" no local onde seria adicionado o nosso link e selecionamos a pagina que desejamos que seja aberta. Para dar um exemplo pratico, clique em mim para acessar a pagina onde ensino a colocar links externos

PARAMETROS

Assim como no Links externos, nos podemos utilizar alguns parametros de valor mais semantico para guiar o Google e ajuda-lo a entender o caminho das paginas. para isso podemos utilizar dois parametros:

REL NEXT
Usamos essa tag para informar que o link que foi utilizado por ela, é a "proxima pagina" que queremos que o usuario acesse. Lembrando que esse parametro é de valor semantico, esteticamente não muda nada em seu site
REL PREV
Ao contrario de "next", o parametro "prev" faz referencia a "pagina anterior". Seria meio que um aviso ao Google que o link que foi utilizado com esse parametro representaria a pagina que teria que voltar apos o acesso

LINKS PARA DOWNLOADS

Para criar um link para download utilizamos a mesma tag de sempre: <a>. Uma observação importante a ser feita é que o arquivo que você quer que seja baixado tem que está tambem na mesma pasta do seu site. Assim como nos links internos e externos, os links para download tem parametros que facilitam no processo, como exemplo o parametro (download="nome do arquivo que vai ser baixado") pra acompanhar a linha de raciocinio, clique em mim para baixar um PDF

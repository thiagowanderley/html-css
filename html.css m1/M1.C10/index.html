Para adicionarmos Links ao nosso site vamos utilizar a tag <a>. Para colocar em pratica, vou adicionar um link em seguida de um site que eu gosto muito, de um jogo de enigma chamado notpron

PARAMETROS

Existe um parametro chamado (rel="nofollow") que seria basicamente você avisar ao nosso amigo Google que o site/link que está sendo acessado não tem seu "aval", não é um site que você conhece. Esse parametro parece um pouco com o "external", porem tem um significado um pouco diferente

Uma dica muito importante para quando você for adicionar links externos em seu site, é preciso utilizar os parametros (target="_blank") para abrir o link adicionado em uma nova aba. É muito importante tambem utilizar o parametro (rel="external") para informar ao site que o link que vai ser aberto vai para fora do seu site.

LISTAS OL E UL

LISTAS ORDENADAS

Lista ordenada serve literalmente para fazer uma lista em ordem crescente, seja ela alfabetica, numerica, ou em numeros romanos. Para usa-la vamos usar duas tags. A primeira é "<ol>", essa tag tem a função de avisar ao sistema que estamos fazendo uma lista. Dentro da tag "<ol>" vamos utilizar a tag <li> dentro de cada item da nossa lista. Ex:

  1. Arrumar a casa
  2. tomar banho
  3. ir pra faculdade

Dentro da tag "ol" podemos colocar o parametro type para alterarmos o seu estilo de ordem, fica mais ou menos assim o comando: <ol type="">. Na lista abaixo vou colocar tudo em ordem alfabetica utilizando o parametro "type"

  1. Arrumar a cama
  2. Tomar banho
  3. Ir para faculdade

Além dessas funções. caso você não queira que sua lista comece pelo numero 1, podemos fazer essa alteração atravez do parametro start dentro tbm da tag "ol", ficando dessa forma:<ol start="5">. Na lista abaixo, ela vai começar pelo número 5. Ex:

  1. Arrumar a casa
  2. Tomar banho
  3. Ir para faculdade

LISTAS NÃO ORDENADAS

Praticamente igual as listas ordenadas, o estilo de lista "não ordenada" tem a mesma função da lista ordenada, sua unica diferença é que ela não tem ordem, não é necessario uma ordem de sequencia. Para reproduzi-la utilizamos praticamente os mesmos comandos da lista ordenada. Porem, no lugar de "ol", vamos usar a tag "ul". Acompanhe no exemplo abaixo:

Lista de compras
  • Pão
  • Ovo
  • Queijo
  • Frutas

LISTA DE DEFINIÇÃO

Uma lista regida pela tag <dl> onde você determina o termo atravez da tag <dt>, e a sua definição atravez da tag <dd>
